# Lux Aeterna

Conscious Relationship Intelligence OS built with Next.js 15, React 19, TypeScript, Tailwind CSS, Framer Motion, shadcn-inspired primitives, Supabase, Clerk, and OpenAI integration scaffolding.

## Run

Use Node 20 or newer.

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` into `.env.local` and add keys for OpenAI, Supabase, and Clerk when enabling live integrations.

## Surfaces

- Landing page at `/`
- Couples dashboard at `/dashboard`
- Shared journal at `/journal`
- AI conflict mediator at `/mediator`
- Compatibility system at `/compatibility`
- Meditation portal at `/meditation`
- Growth center at `/growth`
- Insights engine at `/insights`
- Conscious social at `/social`
- Privacy and trust center at `/trust`
